# PIC2TEXT
The aim of this project is allow users to create ASCII art based on the uploaded photo. 
The user has the choice betweeen two kinds of outputs, a text file ".txt" and a jpg file ".jpg".

# TECHNOLOGIES USED

* HTML
* CSS / SCSS
* JAVASCRIPT
* NODEJS / EXPRESS
* MYSQL
